1. это приложение для учета задач и наград ребенка
1. родитель может добавлять задачи и награды
1. ребенок может выполнять задачи и получать награды
1. стоимость задач и наград измеряется в токенах
1. при регистрации в боте пользователь выбирает роль: родитель или ребенок
1. при регистрации в боте пользователь выбирает семью: создается новая семья или присоединяется к существующей
1. при регистрации в боте пользователь вводит свое имя
1. я не знаю, в рабочем ли состоянии сейчас этот бот, или нет
1. в приложении есть 2 компонента: api и ui в виде телеграм бота
1. api - это сервер, который обрабатывает запросы от телеграм бота
1. ui - это телеграм бот, который взаимодействует с пользователем
1. api и ui взаимодействуют друг с другом через rest api
1. для api нужно написать тесты
